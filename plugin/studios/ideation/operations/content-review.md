**Purpose:** Review published deliverables for accuracy and relevance. Content decays as the world changes.

**Procedure:**
- [ ] Review all deliverables produced by this intent
- [ ] Check factual claims against current sources
- [ ] Identify sections that reference outdated information
- [ ] Flag content that needs updating or retirement

**Signals:**
- Quarterly review cadence
- External changes that invalidate assumptions (market shifts, new research)
- User feedback indicating outdated content
