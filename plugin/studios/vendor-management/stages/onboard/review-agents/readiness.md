**Mandate:** The agent **MUST** verify vendor integration is complete and the organization is ready for operational use.

**Check:**
- The agent **MUST** verify that end-to-end data flows are tested including error handling scenarios
- The agent **MUST** verify that all users have appropriate access and training
- The agent **MUST** verify that escalation paths are documented with named contacts and response expectations
- The agent **MUST** verify that integration documentation is sufficient for ongoing operational support
