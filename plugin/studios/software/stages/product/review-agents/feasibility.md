**Mandate:** Challenge whether the specified behavior is implementable within the technical constraints.

**Check:**
- Specified response times and performance targets are realistic given the data model and infrastructure
- Data contracts are compatible with existing schemas (no breaking changes without migration plan)
- Edge cases have defined behavior, not just "handle gracefully"
- No specification assumes capabilities that would require unreasonable effort to build
