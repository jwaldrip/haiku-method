**Mandate:** Verify the design meets accessibility requirements and does not exclude users.

**Check:**
- Color contrast ratios meet WCAG AA minimum (4.5:1 for text, 3:1 for large text and UI components)
- Touch targets are at least 44px on mobile
- All interactive elements are reachable via keyboard and have visible focus indicators
- Information is not conveyed by color alone
- Screen reader flow is logical and all images/icons have appropriate labels
