**Mandate:** Verify the design is internally consistent and aligns with the project's existing design system.

**Check:**
- All spacing, typography, and color values reference named tokens — no raw hex, px, or magic numbers
- Interactive elements have consistent state coverage (default, hover, focus, active, disabled, error)
- Component naming follows the existing pattern language
- Layout grid and breakpoint behavior is consistent across all screens
