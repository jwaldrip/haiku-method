# Unit 01: Login Screen

Design the full-page login screen with social sign-in buttons and forgot-password link.

## Completion Criteria

- [x] Full-page layout with centered card — app logo at top, social buttons prominent
- [x] Responsive breakpoints: mobile (375px), tablet (768px), desktop (1280px)
- [x] Social buttons: "Sign in with Google" and "Sign in with GitHub" with provider logos
- [x] "Forgot password?" link below the social buttons
- [x] Interactive states specified: default, hover, focus, active, disabled, loading
- [x] Touch targets at least 44px on mobile
- [x] Color contrast meets WCAG AA (4.5:1 for text, 3:1 for large text)
