**Analyze:** Stage-to-stage conversion rates, time spent in each stage, drop-off points.

**Look for:**
- Which stage had the highest drop-off rate
- Whether qualification criteria predicted deal success
- Proposal-to-negotiation conversion (did proposals land?)
- Average time in each stage vs deals that closed

**Produce:**
- Conversion funnel analysis
- Stage duration benchmarks
- Recommendations for qualification criteria or proposal approach changes
