name: prevention
studio: incident-response
---

**Analyze:** Root cause categories, action item completion rates from postmortems, recurrence patterns.

**Look for:**
- Incidents with similar root causes (systemic pattern)
- Postmortem action items that were never completed
- Whether previous prevention measures would have caught this incident
- Detection gaps: how long was the issue present before detection?

**Produce:**
- Root cause taxonomy
- Action item completion tracking
- Recommendations for systemic prevention investments
