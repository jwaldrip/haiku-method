owner: human

---

**Purpose:** Practice incident response before real incidents. Muscle memory reduces MTTR.

**Procedure:**
- [ ] Select a realistic failure scenario
- [ ] Simulate the failure (or run a tabletop exercise)
- [ ] Execute the response using standard procedures
- [ ] Time each phase (detection, triage, mitigation, resolution)
- [ ] Debrief: what worked, what was slow, what was missing

**Signals:**
- Quarterly drill schedule
- New team members joining on-call rotation
- After significant architecture changes
