**Analyze:** Control coverage completeness, gap density across domains, remediation effectiveness.

**Look for:**
- Controls that were harder to evidence than expected
- Gaps that recurred from previous assessment cycles
- Areas where compensating controls were used instead of primary controls
- Remediation efforts that addressed symptoms vs root causes

**Produce:**
- Coverage assessment by control domain
- Recurring gap patterns
- Recommendations for process changes to prevent gap recurrence
